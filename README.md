## Installation

Clone the repo

```bash
  git clone https://github.com/devigor/ui-login
  cd ui-login
```

Install all dependencies and the Husky

```bash
  yarn
  yarn install husky
```

Create the `.env.local` with the keys
```env
NEXTAUTH_URL=
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```

Run the project
```bash
  yarn dev
```
